## สิ่งที่ได้ดำเนินการด้วยตนเอง
- Docker Compose เพื่อให้ container ของทุก service สามารถเริ่มทำงานได้ครบถ้วน
- ทดสอบการทำงานของระบบผ่าน browser, Postman และคำสั่ง curl บน HTTPS
- ช่วยตรวจสอบความถูกต้องของระบบในภาพรวมก่อน screenshots

## ปัญหาที่พบและวิธีการแก้ไข
- เครื่องของส่วนตัวทำใน window powershell เลยจำเป็นต้องเพิ่ม cert.pem key.pem เพื่อให้ nginx ทำงานได้
- พบปัญหาเกี่ยวกับ rate limiting ซึ่งในบางกรณีการทดสอบผ่าน Postman ไม่เร็วพอที่จะทำให้เกิด 429 Too Many Requests เลยใช้ auto click


## สิ่งที่ได้เรียนรู้จากงานนี้
ได้เข้าใจการทำงานของ Docker Compose ในการจัดการหลายบริการภายในระบบเดียว
ได้เรียนรู้วิธีการทดสอบระบบแบบ end-to-end ผ่าน Postman, browser และคำสั่ง curl

## แนวทางการพัฒนาต่อไปใน Set 2
ทดสอบระบบแบบ end-to-end บน cloud เพื่อยืนยันว่าระบบยังทำงานได้ครบถ้วนแม้อยู่ในสภาพแวดล้อมจริง
